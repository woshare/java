## 协议
>协议三要素：
>1）语法：即数据与控制信息的结构或格式
>2）语义：即需要发出何种控制信息，完成何种动作以及做出何种响应
>3）时序：即事件实现顺序的详细说明

>HTTP协议三要素举例：
>语法：http规定了请求报文和响应报文的具体格式
>语义：客户端主动发起的操作称为请求，比如get put post等
>时序：一个请求对应一个响应
