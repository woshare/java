## 协议
>协议三要素：
>1）语法：即数据与控制信息的结构或格式
>2）语义：即需要发出何种控制信息，完成何种动作以及做出何种响应
>3）时序：即事件实现顺序的详细说明

>HTTP协议三要素举例：
>语法：http规定了请求报文和响应报文的具体格式
>语义：客户端主动发起的操作称为请求，比如get put post等
>时序：一个请求对应一个响应

## 负载均衡
>四层负载均衡：基于IP+端口的负载均衡：从传输层开始是，使用“ip + port”接受请求，在转发到对应的服务器

>七层负载均衡：基于虚拟的URL或主机IP的负载均衡，在四层负载均衡的基础上，通过应用层协议实现负载均衡
